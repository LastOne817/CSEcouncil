# 교수님과의 인사

## Info
  - 날짜 : 11/30(월) - 12/4(금)
  - 신임 회장단으로서 교수님들께 인사

## Activity
  - 메일 드리고 찾아가서 인사드리기
  - 교수진 메일은 prof@cse.snu.ac.kr 로 보내면 모든 교수님께 보내짐
  - 답장 주시는 분들만 뵈면 됨 (5~6분 정도)
  - 면담 시간은 케바케

## Etc
  - 임기 초반 일정이 많아서 인사가 늦음
  - 학부장님 : 아예 날 잡아서 이제희교수님, 문봉기교수님, 학부장님 셋과 점심식사 일정을 잡으심. 1시간 좀 넘게 얘기함
  - 김건희교수님 : 20분 정도, 별 얘기 안함
